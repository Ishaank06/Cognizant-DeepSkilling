# Hands-On 5.1.4: Lifecycle Hooks (`blogapp`)

- **Objective:** Understand component lifecycle phases (Mounting, Updating, Unmounting).
- **Implementation:** Created a `Blog` class component demonstrating `constructor`, `componentDidMount`, `componentDidUpdate`, and `componentWillUnmount`. `App.js` toggles the rendering of `Blog` to showcase the unmounting phase and cleanup.
- **Verification:** App successfully builds and runs. Lifecycle logs can be observed in the browser developer console.
